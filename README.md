# HOOKS-TS
keyword: hook、typeScript

## 简介
基于kook与typsScript开发的后台应用demo

## 开发
```
# 克隆项目
git clone https://github.com/zhangwinwin/hook-ts

# 进入项目目录
cd hook-ts

# 安装客户端依赖
npm install

# npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动客户端服务
npm start

# 启动服务端服务
npm run start-server
```
浏览器访问 http://localhost:3001
